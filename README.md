# Water-Tracker
Helps track water intake , runs locally as a web browser bookmark but can be deployed. Uses Python for backend, Flask for API , MySQL for database, React for frontend.bSupports user authentication, real-time stats and comprehensive history
## Features 
1. User registration & authentication (JWT) <br>
2. Water intake tracking (quick-add buttons + custom) <br>
3. Real-time progress visualization <br>
4. Daily history with timestamps <br>
5. Statistics & goal tracking <br>
6. Persistent data in MySQL <br>
## Tech Stack
-> Backend : Flask (Python) + MySQL <br>
-> Frontend : React 18 + Vite <br>
-> Authentication : JWT + bcrypt <br>
-> API :  RESTful with CORS
